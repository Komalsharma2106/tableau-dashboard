# Airline Service & Performance Dashboard

##  Problem Statement
Airlines need to monitor service performance and customer satisfaction across different routes and regions.

##  Objective
To create an interactive Tableau dashboard analyzing airline performance using geographic and service-based metrics.

## Tools Used
- Tableau
- Data visualization techniques

##  Dashboard Highlights
- Regional performance comparison  
- Service quality metrics  
- Customer satisfaction indicators  
- Travel trends by geography  

## Key Insights
- Certain regions showed higher satisfaction levels  
- Service delays impacted customer ratings  
- Travel frequency varied significantly by location  

##  Business Value
Helps airline management:
- Improve underperforming routes  
- Enhance service quality  
- Make customer-focused operational decisions
